# Customer Orders Data Pipeline & Reporting Dashboard

## Project Overview
This project demonstrates an end-to-end data pipeline using SQL Server to import, clean, transform, and analyze customer order data from a CSV file.

## Technologies Used
- SQL Server
- T-SQL
- BULK INSERT
- ETL
- Power BI

## Features
- Import CSV data using BULK INSERT
- Data cleaning and transformation
- Stored procedures for automation
- Query performance optimization using indexing
- Reporting and dashboard visualization

## Project Files
- create_tables.sql
- bulk_insert.sql
- transformations.sql
- stored_procedures.sql
- performance.sql
- orders.csv

## Output
- Cleaned and structured data
- Aggregated reports
- Dashboard for business insights
